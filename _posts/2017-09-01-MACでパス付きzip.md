## MACでパス付きzip

#### Windows時代は毎日のように作っていたzip，MACでは毎回ぐぐってやっていたので一旦メモる

パス無し

```sh
$ zip archive.zip ./file_a.txt ./file_b.txt ./file_c.txt
```

パス付き

```sh
$ zip -e archive.zip ./file_a.txt ./file_b.txt ./file_c.txt
```

ディレクトリを圧縮

```sh
$ zip -r ./directory_name
```

展開

```sh
$ unzip ./archive.zip
```
