## nodebrew verup

- homebrewをアップデート

[2017-09-29-homebrew_verupしたときメモ]({% post_url 2017-09-29-homebrew_verupしたときメモ %})

- node ver確認

```sh
$ node -v
```

- nodebrew ver確認

```sh
$ nodebrew -v
```

- nodebrew インストール済ver一覧

```sh
$ nodebrew list
```

- nodebrew インストール可能ver一覧

```sh
$ nodebrew ls-remote
```

- ver指定してインスコ

```sh
$ nodebrew install-binary 8.6.0
```

- 使用するverに固定する

```sh
$ nodebrew use v8.6.0
```

- node ver確認

```sh
$ node -v
```

- ver指定してアンインストール

```sh
$ nodebrew uninstall 7.2.0
```
