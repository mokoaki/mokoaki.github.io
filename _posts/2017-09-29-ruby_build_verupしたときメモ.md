## ruby-build verup

- Homebrewをアップデート

[Homebrew_verupしたときメモ]({% post_url 2017-09-29-homebrew_verupしたときメモ %})

- RBenv, ruby-buildをアップデート

- rbenv インストール可能ver一覧

```sh
$ rbenv install --list
```

- Suggested build environment

```sh
$ brew install openssl libyaml libffi
```

- ruby install

```sh
$ rbenv install 2.4.2 -v
```

- インストールされているruby一覧

```sh
$ rbenv versions
```

- ruby ver 確定

```sh
$ rbenv global 2.2.0
```

- ruby ver 確認

```sh
$ ruby -v
```

- メモ 削除

```sh
$ rbenv uninstall 2.1.2
```
